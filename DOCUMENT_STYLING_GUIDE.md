## Typography Hierarchy

### Level 1 - Main Title (Cover Page)
- **Size**: 48pt
- **Weight**: Bold
- **Color**: Deep Blue (#1F4788)
- **Alignment**: Center
- **Spacing**: Large (400 after)
- **Example**: "BUSINESS PROPOSAL"

### Level 2 - Section Headings (H1)
- **Size**: 32pt
- **Weight**: Bold
- **Color**: Deep Blue (#1F4788)
- **Border**: Gold bottom border (2pt)
- **Spacing**: 400 before, 300 after
- **Example**: "EXECUTIVE SUMMARY"

### Level 3 - Subsection Headings (H2)
- **Size**: 26pt
- **Weight**: Bold
- **Color**: Deep Blue (#1F4788)
- **Border**: Green left border (2pt)
- **Indent**: 200 left
- **Spacing**: 300 before, 250 after
- **Example**: "Market Analysis"

### Level 4 - Sub-subsection Headings (H3)
- **Size**: 22pt
- **Weight**: Bold
- **Color**: Professional Green (#2E7D32)
- **Spacing**: 200 before, 150 after
- **Example**: "Target Market"

### Body Text
- **Size**: 11pt
- **Weight**: Regular
- **Color**: Dark Gray (#333333)
- **Line Spacing**: 1.5x (360 twips)
- **Spacing After**: 200 twips
- **Example**: "This is the main content paragraph..."

### Bullet Points
- **Size**: 11pt
- **Weight**: Regular
- **Color**: Dark Gray (#333333)
- **Indent**: Proper bullet indentation
- **Spacing**: 280 line spacing, 120 after
- **Example**: "â€¢ Key point or feature"

## Spacing Guidelines

### Vertical Spacing
- **Before H1**: 400 twips (0.28 inches)
- **After H1**: 300 twips (0.21 inches)
- **Before H2**: 300 twips (0.21 inches)
- **After H2**: 250 twips (0.17 inches)
- **Before H3**: 200 twips (0.14 inches)
- **After H3**: 150 twips (0.10 inches)
- **After Body**: 200 twips (0.14 inches)
- **After Bullet**: 120 twips (0.08 inches)

### Horizontal Spacing
- **H2 Left Indent**: 200 twips (0.14 inches)
- **Bullet Indent**: Standard (0.25 inches)
- **Body Text**: No indent

### Line Spacing
- **Body Text**: 1.5x (360 twips)
- **Bullet Points**: 1.4x (280 twips)

## Customization

### Change Primary Color
Edit `lib/documentExport.ts`:
```typescript
const COLORS = {
  primary: '1F4788',  // Change this hex code
  // ... rest of colors
}
```

### Change Accent Color
```typescript
const COLORS = {
  // ...
  accent: 'E8A537',  // Change this hex code
  // ...
}
```

### Adjust Font Sizes
In `parseMarkdownToParagraphs()`:
```typescript
// H1 headings
size: 32 * 2,  // Change 32 to desired size

// H2 headings
size: 26 * 2,  // Change 26 to desired size

// H3 headings
size: 22 * 2,  // Change 22 to desired size
```

### Adjust Spacing
```typescript
// After H1
spacing: { after: 300, before: 400 }  // Change these values

// After body text
spacing: { line: 360, after: 200 }  // Change these values
```
