## Step-by-Step: Adding Images in Word

### Step 1: Download Company Profile
1. Go to Dashboard â†’ Generate tab
2. Click "Company Profile"
3. Fill form with company details
4. Click "Generate"
5. Wait for loading screen to complete
6. Click "Download as Word (.docx)"
7. File downloads to your computer

### Step 2: Open in Microsoft Word
1. Find downloaded file on your computer
2. Double-click to open
3. File opens in Microsoft Word
4. Document shows multiple slides

### Step 3: Navigate to Slide
1. Scroll to the slide where you want to add image
2. Click where you want the image to appear
3. Position cursor in the right location

### Step 4: Insert Image
1. Click "Insert" menu at top
2. Click "Pictures" option
3. Click "This Device" (to upload from computer)
4. Browse to find your image file
5. Click image file
6. Click "Insert" button
7. Image appears in document

### Step 5: Resize Image
1. Click image to select it
2. Drag corner handles to resize
3. Hold Shift while dragging to maintain aspect ratio
4. Make image appropriate size for slide

### Step 6: Position Image
1. Click and drag image to desired position
2. Position next to or below text
3. Adjust spacing as needed
4. Make sure text is still readable

### Step 7: Repeat for Other Slides
1. Scroll to next slide
2. Repeat steps 3-6
3. Continue for all slides
4. Add images to all desired locations

### Step 8: Save File
1. Press Ctrl+S (or Cmd+S on Mac)
2. File saves with images
3. File is now ready to use

## Troubleshooting

### Image Won't Insert
**Problem**: Image doesn't appear when trying to insert
**Solution**:
1. Make sure file is in supported format (JPG, PNG, GIF)
2. Check file size (under 5MB)
3. Try uploading from computer instead of web
4. Try different image file

### Image Too Large
**Problem**: Image takes up too much space
**Solution**:
1. Click image to select
2. Drag corner handle to make smaller
3. Hold Shift while dragging to keep proportions
4. Resize to appropriate size

### Image Too Small
**Problem**: Image is hard to see
**Solution**:
1. Click image to select
2. Drag corner handle to make larger
3. Hold Shift while dragging to keep proportions
4. Make image larger but still readable

### Image Blurry
**Problem**: Image appears blurry or pixelated
**Solution**:
1. Use higher resolution image
2. Use original image instead of copy
3. Check image quality before inserting
4. Try different image file

### Can't Edit Document
**Problem**: Document is read-only
**Solution**:
1. Right-click file
2. Properties â†’ Uncheck "Read-only"
3. Click OK
4. Open file again

### Images Not Saving
**Problem**: Images disappear when saving
**Solution**:
1. Make sure you're using Word (not just viewing)
2. Press Ctrl+S to save
3. Save as .docx format
4. Check if file saved with images
