## Adding Images/Decorations

### How to Add Images

Since the document is generated as Word (.docx), you can easily add images:

1. **Download the generated document**
2. **Open in Microsoft Word or Google Docs**
3. **For each slide:**
   - Click where you want to add an image
   - Insert â†’ Image
   - Choose image from your computer
   - Resize and position as needed

### Image Placement Tips

**Cover Page:**
- Add company logo at top
- Add decorative background elements
- Add company photo at bottom

**Content Slides:**
- Add relevant images next to bullet points
- Add team photos on team slide
- Add achievement images on achievements slide
- Add service images on services slide

### Recommended Image Sizes

- **Cover page logo**: 200x200px
- **Slide images**: 300x400px
- **Team photos**: 150x150px
- **Background elements**: Full slide width

## File Format

### Word Document (.docx)
- **Format**: Microsoft Word compatible
- **Pages**: Multiple slides with page breaks
- **Editable**: Fully editable in Word/Google Docs
- **Images**: Can add images easily
- **Printing**: Print-ready format
- **Sharing**: Easy to share via email

### Text Document (.txt)
- **Format**: Plain text
- **Content**: Text only, no formatting
- **Editable**: Any text editor
- **Images**: Not supported
- **Printing**: Basic text format
- **Sharing**: Universal compatibility
