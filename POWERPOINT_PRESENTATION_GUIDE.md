## Adding Images

### How to Add Images in PowerPoint

1. **Open Presentation**
   - Right-click file
   - Open with PowerPoint

2. **Navigate to Slide**
   - Click on slide where you want image
   - Choose location on slide

3. **Insert Image**
   - Click Insert menu
   - Click Pictures
   - Choose image from computer
   - Click Insert

4. **Resize & Position**
   - Drag corners to resize
   - Drag to reposition
   - Make sure text is still readable

5. **Save**
   - Press Ctrl+S
   - File saves with images

### Image Placement Ideas

**Cover Slide:**
- Add company logo at top
- Add company photo at bottom

**Content Slides:**
- Add relevant images next to bullet points
- Add team photos on team slide
- Add service images on services slide
- Add achievement images on achievements slide

### Image Size Recommendations
```
Cover Logo:        200x200 pixels
Cover Photo:       400x300 pixels
Slide Images:      300x400 pixels
Team Photos:       150x150 pixels
Service Images:    250x300 pixels
```

## Troubleshooting

### PowerPoint Won't Open
**Solution**: Make sure PowerPoint is installed
- Use Microsoft PowerPoint
- Or use LibreOffice Impress
- Or use Google Slides (upload file)

### Images Won't Insert
**Solution**: Check image format
- Use JPG, PNG, or GIF
- Check file size (under 5MB)
- Try different image

### Text Looks Wrong
**Solution**: Check formatting
- Font should be Times New Roman
- Size should be appropriate
- Color should be visible

### Can't Edit Slides
**Solution**: Make sure file is not read-only
- Right-click file
- Properties â†’ Uncheck "Read-only"
- Open in PowerPoint
- Edit and save
